# Travel Agent Assistant

This is an AI-powered application to help travel agents find the best travel options for their clients, considering factors like visa requirements, budget, and travel preferences.

## Features

* **Intelligent Flight Search:** Finds the best flight routes, considering visa requirements and suggesting alternative routes to save time and money.
* **Multi-modal Travel Suggestions:** Recommends alternative travel options like driving to a nearby airport to find cheaper or more direct flights.
* **Comprehensive Travel Search:** (Coming soon) Search for hotels, cars, and activities.
* **Visa Recommendations:** (Coming soon) Provides visa recommendations and facilitates e-visa procurement.
